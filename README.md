### App Directory Structure

```
app
├── Dockerfile
├── docker-compose.yml
├── code
│   └── main.py
│   ├── car_price_prediction.ipynb
│   ├── assets
│       └── home.css
├── dataset
│   └── cars.csv
└── model
    ├── scaler.pkl
    └── selling-price.model
```

### Requirements
- Docker
- Python


### Running the app

Run the following command to run the dash app

```make run```

The app will run at address http://localhost:8050

To stop the app, run the following command

```make stop```
